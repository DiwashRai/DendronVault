## Install
`pip install poetry`

## Setting up new project
`poetry new poetry-demo`

Creates the following content:

```
poetry-demo
├── pyproject.toml
├── README.md
├── poetry_demo
│   └── __init__.py
└── tests
    └── __init__.py
```

## Initialise pre-existing project
`poetry init`

## Install dependencies
`poetry install`
